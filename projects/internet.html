Internet

The internet is a global network that connects computers that communicate with each other using cables to connect one another. It enables the sharing of information, resources, and services.

Web

The Web also known as the World Wide Web is a system of interlinked hyperlinks and multimedia content available on the internet.

How are the internet and Web different?

The internet is the part that connects computers globally, while the Web is the service that operates from the internet, allowing users to access and share information through websites.

ASCII and Unicode

ASCII (American Standard Code for Information Interchange) is a character encoding standard that uses 7 bits to represent characters, allowing for 128 unique symbols, including letters, numbers, and control characters. It is primarily used for English text.

Unicode is a more comprehensive character encoding standard that uses up to 32 bits to represent characters, allowing for over a million unique symbols. It supports a wide range of characters from various languages and scripts, making it suitable for global communication.

What is UTF-8?

UTF-8 (Unicode Transformation Format - 8-bit) is a variable-length character encoding for Unicode that uses one to four bytes to represent characters. It is backward compatible with ASCII and is widely used on the internet due to its efficiency in encoding a vast array of characters while minimizing storage space for common characters.

URL

A URL (Uniform Resource Locator) is a web address that specifies the location of a resource on the internet. It consists of a protocol (like http or https), a domain name, and an optional path to a specific file or page.

Domain Name

A domain name is a human-readable address used to identify a specific website or server on the internet. It is part of a URL and is often associated with a particular organization or individual.

HTTP

HTTP (Hypertext Transfer Protocol) is the protocol used for transmitting web pages and other content over the internet. It defines how messages are formatted and transmitted between a client (browser) and a server.

HTTPS

HTTPS (Hypertext Transfer Protocol Secure) is a secure version of HTTP that uses encryption to protect data transmitted between a client and a server.

IP Address

An IP (Internet Protocol) address is a unique numerical identifier assigned to each device

DNS

DNS (Domain Name System) is a system that translates human-readable domain names into IP addresses, allowing users to access websites using easy-to-remember names instead of numerical addresses.

W3C Recommendation

A W3C (World Wide Web Consortium) recommendation is a standard that has been formally approved by the W3C. These recommendations define the specifications for web technologies such as HTML, CSS, and XML.

Difference between XML, XHTML, HTML, HTML5

XML (eXtensible Markup Language) is a flexible markup language used to store and transport data. It allows users to define their own tags and structure.

XHTML (eXtensible Hypertext Markup Language) is a stricter version of HTML that follows the syntax rules of XML. It is designed to be more consistent and easier to parse.

HTML (Hypertext Markup Language) is the standard markup language used to create web pages. It defines the structure and content of a webpage using predefined tags.

HTML5 is the latest version of HTML that introduces new features and capabilities, such as support for multimedia, improved semantics, and enhanced APIs for web applications.
