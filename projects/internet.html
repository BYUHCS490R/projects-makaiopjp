Internet

The internet is a global network that connects computers that communicate with each other using cables to connect one another. It enables the sharing of information, resources, and services.

Web

The Web also known as the World Wide Web is a system of interlinked hyperlinks and multimedia content available on the internet.

How are the internet and Web different?

The internet is the part that connects computers globally, while the Web is the service that operates from the internet, allowing users to access and share information through websites.

ASCII and Unicode

ASCII Stands for American Standard Code for Information Interchange it's a character encoding standard that uses 7 bits to represent characters, with 128 unique symbols, including letters, numbers, and control characters.

Unicode is more complete character encoding standard that uses up to 32 bits to represent characters, with over millions of unique symbols. It's range of characters are from various languages, making it suitable for world communication.

What is UTF-8?

UTF-8 Unicode Transformation Format - 8 bit is a character encoding for Unicode that uses one to four bytes to represent characters.

URL

A URL refers to Uniform Resource Locator. which is a web address location of a source on the internet. It consists of http or https, a domain name, and a way to a specific file or page.

Domain Name

A domain name is a address used to identify a certain website or server on the internet. It is part of a URL and is often associated with an organization or individual.

HTTP

HTTP known as Hypertext Transfer Protocol is the protocol used for transferring web pages and other information over the internet. It specifies how messages are formatted and transferred between the browser and a server.

HTTPS

HTTPS known as Hypertext Transfer Protocol Secure is a secure version of HTTP that uses encryption to protect data transfering.

IP Address

An IP known as Internet Protocol address is a unique numerical code to identifier each device.

DNS

DNS known as Domain Name System is a system that translates user friendly domain names into IP addresses, so users can access websites using those names instead of numerical addresses.

W3C Recommendation

A W3C known as World Wide Web Consortium recommendation defines the specifications for web technologies such as HTML, CSS, and XML.

Difference between XML, XHTML, HTML, HTML5

XML known as eXtensible Markup Language is used to store and transport data, so users can define their own tags and structure.

XHTML known as eXtensible Hypertext Markup Language is a stricter cleaned-up version of HTML. It combines the syntax rules of XML with the structure of HTML.

HTML known as Hypertext Markup Language is the standard markup language is used to create a web pages. It uses tags to defines things like heading, paragraphs, links, images of a webpage.

HTML5 is the latest version of HTML that introduces new features like multimedia elements for audio and video, support for web applications, and more semantic tags for web applications.
